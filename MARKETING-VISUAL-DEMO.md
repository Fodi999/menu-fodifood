## 🎯 User Flows

### Flow 1: Apply Coupon
```
Cart → See "Masz kupon?" 
     → Type "PIZZA20" 
     → Click "Zastosuj"
     → Validation (800ms loading)
     → ✅ Success: Green badge + discount applied
     → Price updated: 85 zł → 68 zł
```

### Flow 2: Newsletter Signup
```
Main Page → Wait 30 seconds
          → Popup appears
          → Enter email
          → Check consent
          → Click "Zapisz się"
          → ✅ Success: WELCOME10 coupon shown
          → Auto-saved to localStorage
          → Email sent (backend)
```

### Flow 3: Earn Loyalty Points
```
Order 85 zł → See "+85 punktów" in cart
            → Complete order
            → Order delivered
            → Backend: earn_points() API
            → Points added to account
            → Push notification: "Otrzymałeś 85 punktów!"
```
